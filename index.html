Aprendiendo CSS y HTML

HTML

Hay varias cosas que aprender de HTML, como por ejemplo:
<h1> Este es un titulo </h1>
Los titulos hay desde h1 hasta h6, siendo h1 el mas grande y h6 el mas pequeño.
<p> Este es un parrafo </p>
para hacer un salto de linea se usa <br>

CSS

Hay varias cosas que aprender de CSS, como por ejemplo:

Layout y Espaciado

  1. margin: Controla el espacio alrededor de elementos externos. Puede especificarse para los lados superior, derecho, inferior e izquierdo.
  2. padding: Controla el espacio alrededor del contenido de un elemento.
  3. display: Controla cómo se muestra un elemento en la página.
  4. position: Controla cómo se posiciona un elemento en la página.
  5. border: Controla el espacio alrededor del borde de un elemento. Puedes especificar el estilo como solid, dotted, dashed
  6. float: Controla cómo se alinean los elementos en la página.
  7. clear: Controla cómo se comportan los elementos después de un elemento flotante.
  8. visibility: Controla la visibilidad de un elemento.
  9. opacity: Controla la opacidad de un elemento.
  10. overflow: Controla cómo se maneja el contenido que desborda un elemento.
  11. z-index: Controla el orden de apilamiento de los elementos.
